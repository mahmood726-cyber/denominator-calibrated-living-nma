# Cardio HF SGLT2 Example Dataset

This dataset is the first canonical cardio seed for the DCLNMA scaffold.

## Source

The trial-level values were seeded from the local review asset:

- `SGLT2i_HF_REVIEW.html`

The extraction records preserve the local line references used to seed the
canonical files.

## Included trials

- DAPA-HF
- EMPEROR-Reduced
- EMPEROR-Preserved
- DELIVER

## Files

- `registry_records.json`
- `publication_records.json`
- `extraction_records.json`

## Intended use

This seed dataset is suitable for:

- testing the schema and ingestion layer
- linkage validation
- inverse-variance pooling checks
- end-to-end cardio capsule smoke tests

It is not yet a locked publication dataset.
